Este foi um projeto para um curso de Laboratório de Programação II na Universidade de São Paulo. A tarefa era criar uma linguagem de programação simples que pudesse ser usada para programar robôs para que se enfrentassem e coletassem recursos. O jogador pode escrever scripts que definem o comportamento dos robôs e, uma vez esses scripts carregados e atribuídos para cada robô, uma simulação gráfica mostra a batalha resultante.

O programa foi desenvolvido em Java e fez uso do JavaCC para criar um compilador simples para nossa linguagem.
